# Smart Guide - Admin Dashboard

System administration dashboard for Smart Guide platform.

## 🚀 Quick Start

```bash
npm install
npm run dev  # Runs on http://localhost:3001
```

## 📋 Features

- ✅ Admin Authentication
- ✅ User Management (enable/disable/delete)
- ✅ POI Approval System
- ✅ Global Analytics
- ✅ System Dashboard

## 🔐 Admin Credentials

Email: `admin@smartguide.com`
Password: `123456` (default in seed)

## 📁 Structure

```
src/
├── pages/           # Routes
├── components/      # Reusable components
├── lib/            # API & Store
└── styles/         # Global styles
```

## 🎯 Key Pages

- `/` - Home
- `/auth/login` - Admin Login
- `/dashboard` - Main Dashboard
- `/users` - User Management
- `/pois` - POI Review & Approval
- `/analytics` - System Statistics
